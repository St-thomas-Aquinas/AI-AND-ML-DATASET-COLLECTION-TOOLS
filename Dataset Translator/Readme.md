# Kenyan Languages Dataset Translator

## Overview
The Kenyan Languages Dataset Translator is a web-based tool designed to translate datasets between multiple Kenyan local languages. It is particularly useful for machine learning and NLP projects where data is abundant in one language but scarce in another.

---

## Supported Languages
The translator currently supports:
- English  
- Kiswahili  
- Kamba  
- Maasai  
- Dhaluo  
- Luhya  
- Kalenjin  

---

## Live Application
Access the tool here:

🔗 https://dataset-translator.vercel.app/

## Features
- Translate datasets between multiple Kenyan languages  
- Simple and user-friendly interface  
- Supports text-based dataset transformation  
- Helps in balancing multilingual datasets  
- Useful for NLP and AI model training  

---

## Use Case
This tool is ideal for:
- Translating datasets for machine learning models  
- Enhancing low-resource language datasets  
- Academic research in Natural Language Processing (NLP)  
- Local language AI development  

### Example
The tool has been used to:
- Translate a scam detection dataset from **English to Kiswahili** to improve model training on local language data.

---

## How It Works
1. Open the web application  
2. Input or upload your dataset text  
3. Select the source language  
4. Select the target language  
5. Run the translation  
6. Download or copy the translated dataset  

---
