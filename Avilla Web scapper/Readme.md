# Avilla Web Scraper

## Overview
Avilla Web Scraper is a dynamic web scraping tool designed to collect images from websites for use in training computer vision models. It provides a simple and interactive workflow through a Jupyter Notebook environment.

---

## Features
- Dynamic image scraping from websites  
- User-friendly input prompts  
- Customizable number of images to scrape  
- Automatic saving of images into specified folders  
- Ideal for building datasets for machine learning and computer vision  

---

## How It Works
The scraper allows users to:
1. Input a website URL  
2. Specify the number of images to scrape  
3. Choose a folder to save the images  

The tool then automatically extracts and downloads images based on the provided inputs.

---

## Usage Instructions

### Step 1: Open in Google Colab
- Upload or open the notebook in **Google Colab**

### Step 2: Run the Notebook
- Execute all the cells in order

### Step 3: Provide Inputs
When prompted, enter:
- **URL** – The website link to scrape images from  
- **Number of Images** – How many images you want to collect  
- **Folder Name** – The directory where images will be saved  

### Step 4: Download Images
- The images will be automatically saved in the specified folder
